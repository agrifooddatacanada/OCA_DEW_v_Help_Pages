Vérification des données


Le vérificateur de données vous permet de visualiser vos données, de les modifier, de les vérifier selon les règles décrites par le schéma et de les exporter au format Excel ou .csv.

Obligatoire : Ajouter un schéma

Ajoutez le paquet de schéma que vous souhaitez utiliser pour saisir des données ou vérifier des données existantes. Votre paquet de schéma contient toutes les règles que vous utiliserez pour vérifier votre ensemble de données. Vous devez ajouter un schéma de données pour utiliser l'outil.

Après avoir ajouté un schéma, vous pouvez passer directement à l'étape suivante où vous pourrez saisir des données conformément aux règles du schéma.

Optionnel : Ajouter des données

Vous pouvez également choisir d'ajouter des données existantes à l'outil, soit pour continuer à ajouter des données, soit pour vérifier les données que vous avez créées. Vous pouvez ajouter un fichier Excel ou un fichier .csv. Assurez-vous que les en-têtes sont toujours dans la première ligne. Lorsque vous ajoutez un fichier Excel, vous pouvez sélectionner la feuille spécifique que vous souhaitez utiliser pour la vérification.

Cliquez sur Suivant

Après avoir sélectionné et prévisualisé votre schéma (et éventuellement l'ensemble de données), cliquez sur suivant.
