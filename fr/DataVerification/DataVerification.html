Vérification des données


Le vérificateur de données vous permet de visualiser vos données, de les modifier, de les vérifier selon les règles décrites par le schéma et de les exporter au format Excel ou .csv. En survolant le symbole ? dans chaque colonne, vous pouvez consulter les règles spécifiques pour chaque attribut de votre ensemble de données. Si une colonne est grisée, cela signifie qu'elle n'est pas associée à un attribut du schéma et ne peut pas être vérifiée.

Changer de langue

Si votre schéma est décrit dans plusieurs langues, vous pouvez consulter les différentes versions en utilisant le sélecteur de langue.

Afficher uniquement les lignes avec des erreurs

Après avoir vérifié vos données, vous pouvez naviguer parmi les erreurs en sélectionnant d’afficher uniquement les lignes comportant des erreurs (pour différentes fonctionnalités).

Vérifier

Lorsque vous cliquez sur le bouton, vos données sont vérifiées selon les règles décrites dans le schéma. Les cellules surlignées en vert sont celles qui ont réussi le test de vérification, tandis que celles en rouge indiquent un échec de vérification. Vous pouvez corriger les erreurs directement dans votre ensemble de données.

Exporter les données

Aucune donnée n'est sauvegardée lorsque vous travaillez dans le vérificateur de données. Pour récupérer vos données, vous devez les exporter. Lors de l'exportation, vous pouvez choisir de le faire au format .csv ou Excel. En exportant au format Excel, vos données seront placées sur une feuille, et une deuxième feuille décrira le schéma des données. En exportant au format .csv, vous aurez l'option de conserver vos en-têtes originaux ou de les modifier pour qu'ils soient conformes au schéma. Il peut être utile de modifier les en-têtes pour correspondre au schéma afin que l'association des attributs soit automatisée.
